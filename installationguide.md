# Light Browser - Installation Guide

This guide will show you exactly how to set up the environment and run the browser from your terminal.

---

## 1. Prerequisites
Before running any commands, you must have **Node.js** installed on your computer.
* **Download here:** [https://nodejs.org/](https://nodejs.org/)
* **Version:** Download the **LTS** version for stability.

---

## 2. Get the Code
1. Download the ZIP file from this GitHub repository.
2. Extract the ZIP file to a folder (e.g., your Desktop).

---

## 3. Terminal Commands (Step-by-Step)

Open your **Terminal**, **CMD**, or **PowerShell** and run these exact commands in order:
**npm install**
**npm start**
